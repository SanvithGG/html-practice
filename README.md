# HTML Practice

This repository contains my practice files for learning HTML. It includes various examples demonstrating HTML features and elements.

## Contents
- **Attribute.html**: Examples of HTML attributes.
- **Media.html**: Practice with HTML media elements (images, audio, video).
- **list.html**: Various types of HTML lists (ordered, unordered).
- **page.html**: Basic HTML page structure practice.
- **tables.html**: HTML tables practice.
- Assorted media directories used for practice (`Vedios`, `audios`, `gif`, `images`, `hhttp`).

These files serve as my foundational practice for building web pages with purely HTML.
